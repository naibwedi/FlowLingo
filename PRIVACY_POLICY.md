# FlowLingo Privacy Policy

Last updated: April 3, 2026

## Overview

FlowLingo is an Android translation keyboard. When you enable the keyboard and use live preview, the app sends the text currently being drafted to Google Cloud Translation so it can return a translated preview. FlowLingo is designed to avoid storing typed content locally.

## What Data FlowLingo Processes

- The text currently being typed in the active Android text field
- The selected target language pair
- Basic local app preferences such as keyboard feedback toggles

## How Typed Text Is Used

- Typed text is held in memory only while the current input field is active
- Typed text is sent to Google Cloud Translation only to generate the live preview and apply result
- Typed text is not written to local files, Hive, analytics, or app logs by FlowLingo
- When the keyboard session ends or the process restarts, in-memory draft text is discarded

## Third-Party Services

FlowLingo uses Google Cloud Translation to translate the current draft text. That means the text being previewed is transmitted to Google for processing. Testers should avoid entering highly sensitive personal, financial, or health information during internal testing.

## What FlowLingo Stores Locally

FlowLingo stores only product settings needed for the experience, such as:

- selected language pair
- haptic feedback preference
- sound feedback preference

FlowLingo does not intentionally store:

- typed drafts
- translated previews
- conversation history

## Security and Internal Testing

Internal builds may still use development signing, developer API credentials, and non-public tester workflows. Testers should use the app only for evaluation during internal testing.

## Contact

For internal testing questions or privacy concerns, contact the project owner before continuing to use the keyboard.
